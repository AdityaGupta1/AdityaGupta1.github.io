Projects

Mini Minecraft

Minecraft with various unique biomes and shader features, made with C++, Qt, and OpenGL. Includes shader features such as shadow mapping, volumetric fog, screen space reflections, motion blur, and bloom. Terrain features are generated procedurally using noise functions and signed distance functions.

Crystal Machine

Procedurally generated crystals made in Houdini, rendered using Redshift. Changing the seed value of a generator creates an entirely new crystal without any extra user input.

Waluig-IO

A genetic algorithm written in Java to play the original Super Mario Bros, based on the NEAT method. Plays the game and learns from its mistakes to progress farther each time.

About

Hello, I'm Aditya.
