Projects

Mega Minecraft

Minecraft with real-time RTX path tracing and GPU-acclerated terrain generation. Made together with two teammates using C++, CUDA, OptiX, and DirectX. Includes various biomes, infinite procedurally generated terrain, diffuse and specular shading, and a day/night cycle.

GPU Path Tracer

A path tracer written in C++ and CUDA, complete with features like specular reflection and refraction, depth of field, BVH construction and traversal, and denoising using Intel Open Image Denoise. Models in the render came from Sketchfab; attribution is given on the project page.

Crystal Machine

Procedurally generated crystals made in Houdini, rendered using Redshift. Changing the seed value of a generator creates an entirely new crystal without any extra user input.

Waluig-IO

A genetic algorithm written in Java to play the original Super Mario Bros, based on MarI/O by SethBling and the NeuroEvolution of Augmenting Topologies (NEAT) method. Plays the game and learns from its mistakes to progress farther each time.

Mobycraft

A Minecraft mod written in Java to view and manage Docker containers. Presented at DockerCon 2016. See slides here and related podcast here. Was also integrated with Netflix's Titus container management platform.

Minecraft Modding with Forge

Book co-authored with my dad which teaches kids how to modify Minecraft using Java. Published by O'Reilly and translated into various languages.

About

Hello, I'm Aditya.

I'm a graduate student pursuing an MSE in Computer Graphics and Game Technology at the University of Pennsylvania. My passion for computer graphics allows me to combine my strong coding background with my interest in creative technology. I've honed my technical skills through practical experiences at NVIDIA, teamLab, Walt Disney Animation Studios, and Roblox, as well as personal projects.
